# Samsung Homepage Clone

A high-performance, pixel-perfect front-end clone of the official Samsung homepage. This project was built to master modern web architecture, component design, dynamic layouts, and production-ready UI engineering.

---

## 🚀 Tech Stack

- **Framework:** React 19 & Next.js 16 (App Router)
- **Styling:** Tailwind CSS (Utility-first responsive styling)
- **Component Libraries:** 
  - [Shadcn UI](https://shadcn.com) (Accessible, unstyled primitives)
  - [Flowbite](https://flowbite.com) (Interactive core components)
- **Icons:** Lucide React (Clean, vector-based iconography)
- **Deployment:** Vercel

---

## ✨ Key Features

- 📱 **Fluid Responsiveness:** Complete cross-device optimization ranging from ultra-wide desktops to mobile viewports.
- ⚡ **Next-Gen Performance:** Server-Side Rendering (SSR) and automated image optimization via Next.js.
- 🎨 **Modern Design Tokens:** Highly tailored components using Tailwind CSS variables and Shadcn themes to perfectly replicate Samsung's aesthetic.
- 🧩 **Modular Component Architecture:** Highly reusable UI architecture breaking down complex layouts into discrete React components.

## 🤖 AI-Assisted Development Workflow

In alignment with modern software engineering practices, this project utilized an **AI-augmented development workflow** to accelerate production speed, optimize clean code structures, and debug layout conflicts.

### Tools Utilized:
- **Cline (VS Code Extension):** Integrated directly into the workspace to orchestrate multi-file refactoring and component scaffolding.
- **ChatGPT & Google Gemini:** Used as pairs-programming agents to optimize complex Tailwind configurations and troubleshoot responsive edge cases.

### Key Engineering Takeaways:
1. **Prompt Engineering:** Learned to write precise, context-rich structural prompts to generate correct Shadcn component configurations.
2. **Code Code Reviewing:** Practiced strict validation and testing of AI-generated snippets to ensure type safety, semantic HTML, and accessibility compliance.
